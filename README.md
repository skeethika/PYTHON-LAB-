# PYTHON-LAB-
## Overview
This repository contains Python lab programs completed as part of the Python Programming course. The programs demonstrate fundamental programming concepts, problem-solving techniques, and Python syntax.

## Objectives
- Learn Python programming fundamentals.
- Implement basic algorithms and logic.
- Practice problem-solving using Python.
- Understand functions, loops, conditional statements, and file handling.

## Contents
The repository includes programs related to:
- Variables and Data Types
- Input and Output Operations
- Conditional Statements
- Loops
- Functions
- Strings
- Lists, Tuples, Dictionaries, and Sets
- File Handling
- Exception Handling
- Object-Oriented Programming (if applicable)

## Requirements
- Python 3.x
- Any Python IDE (VS Code, PyCharm, IDLE) or Terminal

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/python-lab.git
   ```

2. Navigate to the project directory:
   ```bash
   cd python-lab
   ```

3. Run any Python file:
   ```bash
   python filename.py
   ```

## Learning Outcomes
After completing these lab exercises, students will be able to:
- Write Python programs to solve basic problems.
